This is a simple AI Chatroom that powered by gemini, integrating both UIKit and SwiftUI and implementing cocoapods dependency management tools
in this portfolio 
1. I use ExyteChat and communicate with google gemini API using HTTP request.
2. I use Cocoapod as the dependency manager
3. I integrate SwiftUI to base UIKit App
